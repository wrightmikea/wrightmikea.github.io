1 Docker Tangle NGINX

1.1 Overview

Create a shell session to set up docker environment. Tangle a Dockerfile. Use docker shell session to run docker build. Use docker shell session to start ngnix. Use a shell to access ngnix content. Use a docker shell session to stop the nginx container.

1.2 Setup

1.2.1 set docker environement variables

eval "$(docker-machine env default)"
env | grep DOCKER
mkdir -p build

1.2.2 environment set up

DOCKER_HOST=tcp://192.168.99.100:2376
DOCKER_MACHINE_NAME=default
DOCKER_TLS_VERIFY=1
DOCKER_CERT_PATH=/Users/mwright/.docker/machine/machines/default

1.3 Create a docker image

1.3.1 Generate a simple Dockerfile (Ubuntu bash) in the build directory

FROM ubuntu:14.04
RUN apt-get update && apt-get install -y nginx
RUN echo '*** It is full of stars ***' > /usr/share/nginx/html/index.html
EXPOSE 80

1.3.2 Build the docker image and tag it

docker build -t dockertanglenginx build

1.4 Run the created docker image

1.4.1 Run NGINX

docker stop dtnweb 2>&1 | tail -n 0 
docker rm dtnweb 2>&1 | tail -n 0 
docker run -d -p 8888:80 --name dtnweb dockertanglenginx nginx -g "daemon off;"
